# OpenAria: AI Receptionist

The World's #1 AI Receptionist for Business. Professional, reliable, and limitless customer interaction automation.

View OpenAria at: https://openaria.app

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
